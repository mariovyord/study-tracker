# Study Tracker CLI

Study Tracker CLI is a command-line tool designed to help you track and manage your study goals effectively. Set weekly study goals, log your study sessions, view your progress, and delete goals as neededâ€”all from your terminal.

## Installation

1. Clone the repository:

```bash
git clone <repository-url>
cd study-tracker
```

2. Install dependencies:

```bash
npm install
```

3. Make the script globally accessible:

```bash
npm install -g .
```

## Usage

### Start the CLI

Run the following command to start the tool:

```bash
study
```

### Menu Options

1. Add a New Goal: Define a new study goal by providing:

- Title
- Weekly study hours
- Duration (in weeks)

2. Log a Study Session: Record study hours for an existing goal.

3. View Progress: Check your study progress with:

- Total progress percentage
- Visual progress bars for easy tracking

4. Delete a Goal: Remove a goal from your tracker.

5. Exit: Quit the application.
